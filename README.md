 # PC Hardware Diagnostics & Repair Lab

## Overview
This project demonstrates hands-on diagnosis, troubleshooting, and repair of common personal computer hardware failures. The scenarios reflect real-world issues encountered in educational and enterprise environments.

## Equipment & Tools
- Desktop and laptop PCs
- BIOS/UEFI
- SATA HDD & SSD
- RAM modules
- Multimeter (conceptual)
- Diagnostic software (SMART, POST codes)

## Repair Scenarios

### Scenario 1: Hard Drive Failure
**Symptoms:**
- Slow boot
- Frequent freezing
- SMART warnings

**Diagnosis:**
- Checked BIOS detection
- Ran SMART health check
- Verified SATA connections

**Solution:**
- Replaced HDD with SSD
- Reinstalled OS
- Verified system stability

---

### Scenario 2: Faulty RAM
**Symptoms:**
- Random restarts
- Blue screen errors

**Diagnosis:**
- Tested RAM slots individually
- Verified POST error behavior

**Solution:**
- Removed defective RAM module
- Verified system passed boot tests

---

### Scenario 3: Power Supply Failure
**Symptoms:**
- PC does not power on
- No fan activity

**Diagnosis:**
- Verified outlet and power cable
- Used multimeter concept to test voltage rails

**Solution:**
- Replaced PSU
- Verified stable power delivery
